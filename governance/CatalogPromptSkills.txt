-- A. DB & SCHEMA (9) ------------------------------------------

## db-schema-guardian
Task types: schema governance, tabulky, sloupce, indexy, BaseGuid, migrace
Description: Autoritativní správce DB schématu. Zajišťuje, že všechny tabulky mají povinné
  sloupce (Guid, CreatedAt, UpdatedAt, Emoji, Colors, scoring sloupce), správné indexy,
  FK ve formátu SomethingGuid a BaseGuid pro multi-type entity. Řídí migrace bezpečně.
Inputs: název tabulky nebo seznam tabulek, požadované schéma změny
Outputs: SQL migration script, schema diff report, bezpečnostní hodnocení migrace
Limitations: nespouští migrace automaticky — vždy vyžaduje potvrzení operátora

## db-migration-validator
Task types: validace SQL migrací, safety check před spuštěním
Description: Analyzuje SQL migration skripty před spuštěním. Detekuje destruktivní operace
  (DROP, TRUNCATE, ALTER bez NULL default), chybějící sloupce, porušení FK constraints.
Inputs: SQL migration script, aktuální schema
Outputs: seznam rizik (CRITICAL/WARNING/INFO), doporučení oprav
Limitations: nenahrazuje testování na dev DB — vždy otestovat před produkcí

## db-null-audit
Task types: NULL analýza, datová kvalita, prioritizace backfillu
Description: Skenuje všechny tabulky a reportuje počty NULL hodnot per sloupec seřazené
  dle NULL ratio DESC. Identifikuje highest-priority kandidáty pro backfill.
Inputs: seznam tabulek nebo ALL, volitelně min NULL ratio threshold
Outputs: NULL audit report (tabulka, sloupec, NULL count, NULL ratio, priority rank)
Limitations: max 200 tabulek na run; neanalyzuje JSONB vnořené hodnoty

## db-index-optimizer
Task types: indexy, výkon, missing/redundant indexes
Description: Identifikuje chybějící indexy na FK a filter sloupcích (WHERE, ORDER BY).
  Detekuje duplikátní a redundantní indexy. Navrhuje CREATE/DROP INDEX příkazy.
Inputs: seznam tabulek nebo ALL, query log (volitelně)
Outputs: doporučení CREATE INDEX, DROP INDEX, estimated performance gain
Limitations: navrhuje pouze, nespouští — DBA potvrzení povinné

## db-constraint-checker
Task types: FK constraints, unique constraints, check constraints
Description: Validuje všechny FK, UNIQUE a CHECK constraints v databázi.
  Reportuje porušení a osiřelé FK záznamy (parent neexistuje).
Inputs: seznam tabulek nebo ALL
Outputs: seznam violations s tabulka/sloupec/hodnota, doporučení oprav
Limitations: pouze čte; neopravuje data automaticky

## db-table-health-monitor
Task types: monitoring tabulek, row counts, freshness, bloat
Description: Monitoruje zdraví tabulek: počty řádků, datum posledního UpdatedAt,
  odhadnutý table bloat. Flaguje stale tabulky překračující freshness SLA.
Inputs: seznam tabulek, SLA konfigurace (ImportantScore → max stale hours)
Outputs: health report per tabulka (rows, last_update, bloat_estimate, sla_status)
Limitations: bloat estimate pouze přibližný bez VACUUM ANALYZE

## db-column-score-auditor
Task types: scoring sloupce, ConfidenceScore, ReliabilityScore, QualityScore
Description: Ověřuje, že scoring sloupce (ConfidenceScore, ReliabilityScore, QualityScore)
  jsou populovány ve všech záznamech. Reportuje NULL violations per tabulka.
Inputs: seznam tabulek nebo ALL
Outputs: NULL count per scoring sloupec, seznam Guidů s chybějícím score
Limitations: max 100 oprav na run; neopravuje skóre bez zdroje dat

## db-foreign-key-resolver
Task types: dangling FK, osiřelé záznamy, referenční integrita
Description: Hledá FK sloupce (SomethingGuid) odkazující na neexistující parent záznamy.
  Navrhuje buď opravu (správný Guid) nebo soft-delete osiřelého záznamu.
Inputs: tabulka, seznam FK sloupců
Outputs: seznam dangling FK s Guid a návrhem opravy, SQL fix script
Limitations: neopravuje automaticky; vždy vyžaduje review

## db-view-dependency-tracker
Task types: view dependencies, schema změny, bezpečné migrace
Description: Mapuje závislosti analytických views na tabulkách a sloupcích.
  Před schema změnou ověří, které views budou ovlivněny a potřebují rebuild.
Inputs: název tabulky nebo sloupce ke změně
Outputs: dependency graph, seznam dotčených views, pořadí rebuild
Limitations: neanalyzuje dynamické SQL ani stored procedures
