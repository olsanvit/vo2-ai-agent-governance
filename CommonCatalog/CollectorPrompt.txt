## Web Search Protocol

Rate limits:
- max 20 search queries per run
- min 1s delay mezi queries
- max 5 sources per category per run

Search strategie:
1. Prioritizovat kategorie s nejnižším count
2. Pro každou prioritní kategorii: 2-3 search queries s různými formulacemi
3. Kombinovat obecné i specifické queries

Search queries doporučené formulace:
- "{category} wallpaper high resolution free"
- "{category} {target_type} background 4K"
- "free {category} wallpaper download"
- site:unsplash.com OR site:pixabay.com OR site:pexels.com "{category}"

Doporučené zdroje (public, vysoká kvalita):
Tier A: unsplash.com, pixabay.com, pexels.com, wallhaven.cc
Tier B: wallpaperscraft.com, hdwallpapers.in, wallpaperflare.com
Tier C: ostatní veřejné weby

## Classification Rules

Pro každý nalezený výsledek:
1. Určit target type (Mobile / Desktop) dle rozměrů nebo kontextu
   - Mobile: portrait orientation (výška > šířka) nebo označení "mobile/phone"
   - Desktop: landscape orientation (šířka > výška) nebo označení "desktop/PC"
   - Pokud nelze určit: skip nebo uložit s target_type="unknown" pro manuální review
2. Zařadit do kategorie z povolených category listů
   - Exact match nebo nejbližší kategorie
   - Pokud žádná kategorie nesedí: skip + log "⏭️ Skipped: no matching category"
3. Quality check (viz Ch 7)

## Balance Priority

Výpočet priority kategorie:
- priority_score = max_count - category_count
- Kategorie s nejvyšším priority_score = první na řadě
- Cíl: rovnoměrné pokrytí přes všechny kategorie

Log: "⚖️ Balance: {category} má {count} položek (priority: {priority_score})"

## Idempotency

Před zápisem vždy zkontrolovat:
- run_select_sql("SELECT 1 FROM {TargetTable} WHERE SourceUrl='{url}' OR ImageUrl='{imageUrl}'")
- Pokud existuje: skip + log "⏭️ Duplicate: {url}"
- Pokud neexistuje: upsert

## Canonical Database Standard

Stejný standard jako CatalogPrompt/ManagerPrompt:
- Guid uuid PRIMARY KEY DEFAULT gen_random_uuid()
- CreatedAt timestamptz NOT NULL DEFAULT now()
- UpdatedAt timestamptz NOT NULL DEFAULT now()
- Emoji text NOT NULL DEFAULT ''
- Colors text NOT NULL DEFAULT ''
- + 22 scoring sloupců numeric(6,3)

## Collection Items Schema

Doporučené schéma pro collector tabulky (PascalCase):

Shared columns pro všechny collector tabulky:
- Guid uuid PRIMARY KEY
- CollectionType text NOT NULL  — "Mobile" | "Desktop" | "Icon" | "Font" | ...
- Category text NOT NULL         — kategorie dle category listu
- SourcePageUrl text             — URL stránky kde byl nalezen
- ImageUrl text                  — přímá URL obrázku (pokud dostupná)
- SourceLabel text               — název/popis zdroje
- Title text                     — název wallpaperu/souboru pokud znám
- Width integer                  — šířka v pixelech
- Height integer                 — výška v pixelech
- FileFormat text                — jpg | png | webp | svg | ...
- FileSizeBytes integer          — velikost souboru pokud známa
- License text                   — licence (public_domain | cc0 | cc_by | unknown)
- SourceName text                — Unsplash | Pixabay | Pexels | ...
- ConfidenceScore numeric(6,3)   — kvalita/relevance score
- IsApproved boolean DEFAULT false — manuální schválení
- Tags text[]                    — tagy/keywords
- Metadata jsonb
- IsDeleted boolean DEFAULT false
- CreatedAt, UpdatedAt, Emoji, Colors + scoring sloupce

Business key: SourcePageUrl + CollectionType + Category (UNIQUE)
Alternativa: hash(ImageUrl) pokud ImageUrl dostupná

CHAPTER 6: QUALITY & VALIDATION

## Quality Filter

Odmítnout pokud:
- HTTP status != 200 na image URL
- Rozměry < 800px (Mobile) nebo < 1920px šířka (Desktop) — pokud rozměry známy
- Paywall nebo login required
- Watermark přes celý obrázek (pokud detekovatelné)
- Unsafe content (explicitní, násilný, nelegální obsah)
- Nesedí do žádné povolené kategorie
- SourceUrl nebo ImageUrl již existuje v DB (duplicate)

Přijmout s nižším ConfidenceScore:
- Rozměry pod optimem ale nad minimem
- Zdroj Tier C (nižší reliability)
- Kategorie fuzzy match (ne exact)

ConfidenceScore výpočet:
- Source Tier A: base 85
- Source Tier B: base 70
- Source Tier C: base 55
- Exact category match: +10
- Direct ImageUrl available: +5
- High resolution confirmed: +5
- License known: +5

## Duplicate Detection

Primární: SourcePageUrl match
Sekundární: ImageUrl match (pokud dostupná)
Terciární: hash(ImageUrl) v SharedImages (pokud image byl stažen)
